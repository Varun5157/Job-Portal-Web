# Job-Portal-Web
A simple and beginner-friendly Spring Boot REST API project for managing job postings. This project demonstrates how to build backend applications using Java, Spring Boot, and a layered architecture (Controller → Service → Repository → Model).

Features :-

Fetch all available job posts,

Add new job postings,

Store and manage job data using an in-memory list,

Clean and modular Spring Boot project structure,

Easy to extend into a full-stack or database-powered application.



Tech Stack :-

Java 17+

Spring Boot

Spring Web

Maven





How to Run :-

Clone the repo

Open in any Java IDE (IntelliJ/Eclipse/Spring Tool Suite)

Run the Spring Boot application

Access the API using Postman or browser
